## Key Features

### **🎯 Unified Search**
- One search box
- Queries both platforms simultaneously
- 30-60 second results

### **📊 Source Cards**
- Shows what was found from each platform
- Google Maps: Store info
- Amazon: Product info

### **📋 Concise Summary**
- 5 key metrics at a glance
- Total reviews
- Average rating
- Sentiment breakdown

### **🤖 Detailed Summary**
- Sentiment pie chart
- Statistics breakdown
- AI-generated insights:
  - Key themes
  - Strengths
  - Pain points
  - Recommendations

### **📝 Filterable Reviews**
- View all reviews (Google + Amazon)
- Filter by sentiment
- Source badges
- Verified purchase badges

### **💬 Interactive Chat**
- Ask questions about reviews
- GPT-4o powered
- Instant answers

## Technical Stack

**Frontend:**
- React 18.3.1
- Vite 5.4.1
- React Router Dom
- Lucide React (icons)
- Recharts (visualization)
- Axios (HTTP client)

**Backend:**
- Flask 3.0.0
- Python 3.13
- Google Places API
- OpenAI GPT-4o API
- Selenium (web scraping)
- TextBlob (sentiment analysis)

**Database:**
- None (stateless API)

## Setup & Run

### **1. Backend**
```bash
cd backend
python app.py
```
Runs on: http://localhost:5000

### **2. Frontend**
```bash
cd frontend
npm run dev
```
Runs on: http://localhost:5173

### **3. Access**
Open browser: http://localhost:5173
